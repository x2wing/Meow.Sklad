3 типа участников:
- поставщик
- доставщик
- заказчик

При регистрации:

- поставщик
	- вводит количество складов
	Для каждого склада:
		- описание (адрес)
		- список товаров
		Для каждого товара:
			- описание
			- категория, подкатегория
			- тип (сыпучий, опасный и т. п.)
			- фото, видео
			- количество


- доставщик
	- количество гаражей
	Для каждого гаража:
		- описание (адрес)
		- список машин
		Для каждой машины:
			- грузоподьёмность
			- тип перевозимого груза (сыпучий, опасный и т. п.)
			- цена за километр на хорошей дороге


- заказчик
	- вводит количество пунктов приёма
	для каждого пункта приёма:
		- описание (адрес)
		- список необходимых товаров
		Для каждого товара:
			- объём (количество)
